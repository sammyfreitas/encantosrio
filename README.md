# 🌎 Encantos Rio - Sistema de Turismo

![Java](https://img.shields.io/badge/Java-17-blue)
![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3.x-green)
![MySQL](https://img.shields.io/badge/MySQL-Database-orange)
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)

Sistema completo de gestão de turismo desenvolvido como projeto final do curso de Programador Back-End.

A aplicação permite o cadastro, consulta, edição e exclusão de clientes, passeios e vendas, integrando um front-end web com uma API REST desenvolvida em Java com Spring Boot e banco de dados MySQL.

✔ Atualização de registros
